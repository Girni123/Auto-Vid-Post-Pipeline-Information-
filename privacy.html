Privacy Policy for AI Content Flow

Effective Date: July 3, 2026

1. General Information

"AI Content Flow" is a private, non-commercial internal automation utility developed and used exclusively by the account owner. This application is not available to the public and does not offer commercial services.

2. Data Collection and Usage

Our application integrates with the TikTok API using the following scopes to streamline the developer's personal content creation workflow:

  • user.info.basic: Used solely to identify and verify the connected TikTok creator account.
  • video.publish: Used exclusively to upload and publish pre-rendered video files directly to the authorized user's TikTok profile.

3. Data Storage and Security

We respect your privacy. Because this is a private internal tool:

  • No personal user data, profiles, or analytics are collected, processed, or stored on external servers.
  • All API Access Tokens and authentication credentials are encrypted and stored locally within the private environment of the developer.
  • We do not share, sell, or distribute any data to third parties.

4. Contact

For any questions regarding this private application, please contact the developer via the official repository or contact information provided on the main landing page.
