AI Content Flow

Private Content Pipeline Automation

An internal, private utility tool designed to streamline creative workflows. It securely connects a single creator's TikTok profile via OAuth 2.0 to automate the scheduling, direct publishing, and management of pre-rendered, high-quality AI short-form videos.

Privacy Policy

Last updated: July 3, 2026

Overview

AI Content Flow is a private, non-commercial tool built for internal use. It is not a public service and does not operate as a commercial data platform of any kind.

TikTok API Access

To enable automated publishing, the tool requests the following TikTok API scopes:

  • user.info.basic — used solely to identify the authorized creator account connected via OAuth 2.0.
  • video.publish — used to upload pre-rendered video files directly to the connected user's TikTok feed.

Data Handling

We do not collect, store, share, or sell any personal user data to third parties. All authentication tokens are encrypted and used solely for the intended publishing automated workflow by the account owner.
